# 시작

많은 블로그 플랫폼이 있지만 직접 만드는 것보다 자유도가 높을 수는 없을 거 같습니다.  
그래서 Next.js를 사용해서 블로그를 직접 만들기로 했습니다.
<br/>


# Next.js

Next.js는 **Vercel**에서 개발한 React 프레임워크입니다.  
서버 사이드 렌더링(SSR)과 정적 사이트 생성(SSG)을 통해 **SEO 최적화**가 잘 이루어져 있습니다.  
그래서 검색 엔진이 콘텐츠를 더 잘 인식할 수 있다는 블로그로서의 이점이 있습니다.
<br/>


# 프로젝트 생성

Next.js 프로젝트를 생성하기 위해서는 [Node.js](https://nodejs.org/en)가 시스템에 설치되어 있어야 합니다.  
본문은 22.11 버전을 사용하고 있습니다.

프로젝트를 생성할 디렉토리에서 터미널을 열고 다음과 같은 코드를 입력합니다.
```
npx create-next-app@latest
```

그럼 다음과 같이 프로젝트에 대한 설정을 물어보는데, 취향껏 선택하면 될 거 같습니다.

## MDX란?

MDX는 마크다운에 JSX를 추가한 포맷입니다. 마크다운의 간단함과 React 컴포넌트의 유연성을 모두 활용할 수 있습니다.
